# 实战项目入口

这个目录放的是**当前正在共同推进的实战项目**。

它和参考实现的区别是：

- `06-reference-implementations/`
  更偏“标准答案 / 对照样例”
- `07-live-projects/`
  更偏“真实协作中的当前项目”

也就是说，这里的项目通常不是只为了展示，而是为了：

- 和用户一起定义
- 和用户一起迭代
- 保留真实协作痕迹
- 作为后续训练和读码材料

## 建议怎么用

如果你现在的目标是：

- 想看一个正在迭代中的项目怎么组织
- 想理解“自然语言需求 -> 文件结构 -> 代码实现”的过程
- 想看真实协作下的 Agent 项目如何一点点长出来

那这个目录比参考实现更贴近真实开发。

## 推荐阅读顺序

1. 先看项目内 `README.md`
2. 再看项目的读码指南或说明文件
3. 再进入 `schema / tool / agent / workflow / main`
