# Contributing

感谢你关注这个课程仓库。

这个项目的目标不是只堆更多文档，而是持续把它建设成一套真正可学、可用、可扩展的 Agent 学习系统。

## 适合贡献的方向

- 扩写某个 Stage 的知识讲解
- 增加新的引导项目或课后项目
- 补充练习题、案例库、错题库
- 增加更好的前端原型示例
- 修正文档错误、导航问题或表述不清的地方

## 贡献原则

- 优先增强学习闭环，而不是只增加信息量
- 优先保持阶段梯度清楚
- 优先解释“为什么这样设计”
- 新增内容应尽量对应课程中的某个 Stage 或某个项目

## 提交建议

- 保持提交聚焦，一个提交尽量只解决一类问题
- 文档改动优先保持 Markdown 结构清晰
- 如果新增项目，尽量同时补齐：
  目标、知识点、步骤、课后任务、验收标准

## 对课程结构的建议

如果你要新增内容，建议先看：

- [README.md](./README.md)
- [01-curriculum/learning-loop.md](./01-curriculum/learning-loop.md)
- [03-projects/ai-project-teaching-protocol.md](./03-projects/ai-project-teaching-protocol.md)

## Issue / PR 建议

如果后续使用 GitHub Issues / PR，建议在描述中写清：

- 你改的是哪个 Stage 或哪个目录
- 你希望解决什么问题
- 你的改动如何帮助学习者更好学习

## 风格建议

- 尽量清晰、具体、可执行
- 少写空泛口号，多写真实场景和判断标准
- 保持课程的阶段感、项目感和实践感
