# 项目目录

下面这些项目会在后续迭代中继续扩写成更完整的教学案例包。当前先建立项目池，保证课程的广度、深度和难度梯度。

## A. 入门项目

### 1. FAQ 助手

- 难度: 低
- 主线: Agent Foundations + LangChain
- 学习重点: 提示词边界、结构化输出、单轮到多轮过渡

### 2. 结构化信息抽取助手

- 难度: 低
- 主线: LangChain
- 学习重点: structured output、数据格式、可靠性

### 3. 单工具研究助理

- 难度: 低到中
- 主线: LangChain
- 学习重点: 工具调用、上下文组织、结果汇总

## B. 核心进阶项目

### 4. 文档知识问答 Agent

- 难度: 中
- 主线: LangChain + RAG
- 学习重点: 检索、知识边界、答案可信度

### 5. 多工具个人工作助理

- 难度: 中
- 主线: LangChain + MCP
- 学习重点: 工具生态、权限、任务编排

### 6. 带人工审批的流程 Agent

- 难度: 中到高
- 主线: LangGraph
- 学习重点: interrupts、审批流、状态可见性

### 7. 多阶段研究与汇报 Agent

- 难度: 高
- 主线: LangGraph
- 学习重点: 分阶段执行、失败恢复、信息汇总

## C. 复杂任务项目

### 8. 深度研究 Agent

- 难度: 高
- 主线: Deep Agents
- 学习重点: planning、subagents、context engineering、输出质量

### 9. 创作者内容工作台

- 难度: 高
- 主线: Deep Agents + Frontend
- 学习重点: 长任务、内容策划、多面板交互

### 10. 代码库理解与执行顾问

- 难度: 高
- 主线: Deep Agents + MCP + Skill
- 学习重点: 文件系统理解、工具链、复杂目标分解

## D. 系统型项目

### 11. 企业知识与流程一体化 Agent

- 难度: 高
- 主线: LangGraph + MCP + RAG + Backend
- 学习重点: 知识层、权限层、工作流整合

### 12. 销售支持 Agent

- 难度: 高
- 主线: CRM 集成 + Frontend + 审批
- 学习重点: 系统集成、前端体验、业务规则

### 13. 学习教练 Agent

- 难度: 高
- 主线: LangChain / LangGraph / Deep Agents 综合
- 学习重点: 个性化、记忆、课程规划、反馈闭环

### 14. 个人操作系统型 Agent

- 难度: 很高
- 主线: 综合毕业项目
- 学习重点: 长期记忆、多工具、多场景、自定义 UX

## 项目设计原则

- 同时覆盖后端与前端
- 同时覆盖知识、工具、状态与交互
- 同时训练系统能力与产品思维
- 从最小原型走向完整产品
