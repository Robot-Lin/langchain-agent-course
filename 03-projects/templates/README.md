# 项目模板

这里放课程统一使用的项目模板。

- [guided-project-template.md](./guided-project-template.md)
- [homework-project-template.md](./homework-project-template.md)

## 这些模板现在额外强调什么

除了项目目标、步骤和验收标准之外，模板现在还强制要求：

- 每个项目都要写出知识点到功能再到文件或代码的映射
- 每个跟学项目都要包含代码导读要求
- 每个课后项目都要包含代码理解与审核标准
- 学习者不能只看最终结果，必须逐步学会解释关键实现逻辑

## 怎么用

- guided-project-template.md 用于跟学项目
- homework-project-template.md 用于课后独立项目

跟学项目强调讲解、引导和读码，课后项目强调需求、实现、审核与复盘。
