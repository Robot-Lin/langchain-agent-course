# 全栈毕业项目模板

这是课程终局项目的模板入口。它对应 Stage 7 和 Stage 8。

## 项目必须覆盖的模块

- 前端交互层
- Agent runtime 层
- 工具或 MCP 层
- 知识层或数据层
- 评测与观测层
- 部署与产品边界

## 建议的构建顺序

1. 用户场景与核心任务
2. 最小 Agent 主路径
3. 工具与知识接入
4. 状态、审批与记忆
5. 前端控制台
6. 评测与观测
7. 部署与演示
8. 复盘与下一版规划

## 推荐毕业项目包应包含

- 需求说明
- 用户旅程
- 架构图
- 交互稿
- 原型说明
- 评测方案
- 风险清单
- 复盘报告
