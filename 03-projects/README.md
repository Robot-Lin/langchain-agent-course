# 项目库入口

项目库负责把知识变成能力。

建议做法:

- 每个阶段至少做一个“最小项目”
- 每两个阶段做一个“综合项目”
- 在毕业阶段完成一个“产品型项目”
- 每个阶段都配一类“跟学项目”和一类“课后独立项目”
