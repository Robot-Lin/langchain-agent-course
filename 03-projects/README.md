# 项目目录入口

这个目录负责把“知识”变成“能力”。课程里的项目不是附属练习，而是主教学机制的一部分。

## 建议怎么用

- 每个阶段至少完成 1 个跟学项目。
- 每个阶段至少完成 1 个课后独立项目或 challenge。
- 在毕业阶段，把前面学到的内容收束成一个产品型项目。
