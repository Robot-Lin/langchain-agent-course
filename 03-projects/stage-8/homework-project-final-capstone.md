# 毕业项目: 最终综合项目

## 所属阶段

Stage 8: Capstone Mastery

## 需求描述

请独立完成一个“最终综合 Agent 项目”方案。你需要从课程前 7 个阶段中选择并整合必要能力，设计一个从后端到前端的 Agent 产品。它必须有明确用户、明确场景、完整主路径、综合架构、评测与治理方案、交付思路，以及一份答辩和复盘材料。

## 约束条件

- 必须覆盖前端与后端
- 必须覆盖 Agent runtime 与知识或工具层
- 必须覆盖评测、追踪和治理
- 必须有可演示主路径
- 必须有一份下一版规划
- 不要求你真的把所有代码全部实现

## 不直接给出的内容

- 不直接给最终题目答案
- 不直接给完整系统图
- 不直接给答辩答案

这些都需要你先自己完成。

## 你需要自己完成的部分

- 确定题目与用户场景
- 写出主路径
- 画出综合系统图
- 写出前后端与 runtime 协同方式
- 写出知识或工具接入方式
- 写出评测、追踪和治理方案
- 写出交付与演示思路
- 写出答辩提纲
- 写出复盘和下一版规划

## 最终效果标准

当别人读完你的方案后，应该能明确看见以下结果:

- 这是一个真实可讨论的 Agent 产品，而不是拼凑概念
- 它的用户、场景和价值清楚
- 它的主路径完整且可演示
- 它的架构覆盖了课程核心能力层
- 它知道如何被评测、追踪和治理
- 它有清晰的下一版方向

## 验收标准

你的方案至少需要满足:

- 有明确题目与用户场景
- 有一条完整主路径
- 有一张综合系统图
- 有前后端与 runtime 协同说明
- 有知识或工具层说明
- 有评测、追踪和治理方案
- 有交付与演示说明
- 有答辩提纲
- 有复盘与下一版规划
- 有一段 300 字以上的综合项目说明

## 如何让 Codex 辅导你

你可以对 Codex 说:

- 我现在做的是 Stage 8 毕业项目，请不要直接给我完整答案
- 你只负责检查我的题目、主路径、系统图、答辩提纲和复盘
- 最后按毕业验收标准帮我评分，并指出我哪里还没有真正形成方法闭环

## 建议复盘问题

- 我的题目是否真的值得做
- 我的方案是否覆盖课程核心能力
- 我的答辩是否能经得起追问
- 我的下一版规划是否真正聚焦
