# 引导项目: 毕业项目框架

## 所属阶段

Stage 8: Capstone Mastery

## 项目目标

为你的毕业项目建立完整框架。它需要明确选题价值、用户场景、主路径、综合架构、演示方案、答辩材料和复盘结构。

## 对应知识点

- 选题三问法
- 覆盖检查法
- 主路径优先法
- 答辩准备法
- 复盘四问法

- 一份毕业项目立项说明
- 一张综合系统图
- 一份主路径说明
- 一份答辩提纲
- 一份复盘模板

## AI 带学步骤

### Step 1: 做选题判断

要做什么:

- 明确用户、价值和 Agent 必要性

为什么先做:

- 题目不成立，后面做再完整也没有意义

知识点关联:

- 选题三问法

### Step 2: 做课程覆盖检查

要做什么:

- 检查这个题目是否覆盖前 7 个阶段核心能力

为什么先做:

- 毕业项目必须体现综合掌握

知识点关联:

- 覆盖检查法

### Step 3: 定义主路径

要做什么:

- 明确最核心演示路径

为什么先做:

- 毕业项目首先要可演示、可解释

知识点关联:

- 主路径优先法

### Step 4: 准备答辩结构

要做什么:

- 把项目讲述结构搭起来

为什么先做:

- 做得出来和讲得明白是两件事

知识点关联:

- 答辩准备法

### Step 5: 准备复盘框架

要做什么:

- 提前定义如何复盘和规划下一版

为什么先做:

- 毕业不是终点，而是下一轮迭代起点

知识点关联:

- 复盘四问法

## 阶段检查点

- 能说清选题价值
- 能证明题目覆盖课程核心能力
- 能说明主路径
- 能准备答辩与复盘框架

## 与 Codex 的配合方式

推荐对 Codex 说:

- 先根据 Stage 8 的概念文件，带我一步步搭好毕业项目框架
- 每一步先解释对应知识点，再帮我检查题目和结构
- 不要一开始就帮我做完整毕业方案，先让我自己选题和定主路径
