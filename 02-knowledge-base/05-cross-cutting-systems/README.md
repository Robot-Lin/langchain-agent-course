# Cross-cutting Systems

这一层负责把 Agent 从概念项目升级为真实系统。

## 课程会覆盖的横向能力

- Tools
- Structured output
- Memory
- MCP
- RAG
- 数据库与存储
- 后端 API
- Observability 与 evaluation
- Guardrails
- Skill 化能力封装

## 每个主题解决什么问题

- Tools: 让 Agent 不只会回答，还能行动
- Structured output: 让输出可消费、可验证、可进入后续系统
- Memory: 让系统具备跨轮和跨任务连续性
- MCP: 让 Agent 以标准协议接入外部能力
- RAG: 让 Agent 在受控知识范围内工作
- 数据层: 让状态、知识和业务数据有可靠落点
- API: 让 Agent 真正接入业务系统
- Observability / evaluation: 让你知道系统哪里好、哪里坏
- Guardrails: 让系统更安全、更可控
- Skill: 让高频能力可以复用和封装
