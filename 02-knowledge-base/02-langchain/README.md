# LangChain

这一部分聚焦 LangChain 作为高层 Agent 开发入口的角色。

## 为什么先学 LangChain

- 它能让你更快搭出可工作的高层 Agent 原型
- 它适合新手建立“组件化”理解
- 它把模型、tools、structured output、middleware 等常见问题整理得比较清楚

## 这里会覆盖什么

- 高层 Agent 原型
- Tools 与 toolkits
- Structured output
- Middleware
- 基础 human-in-the-loop
- 高层 RAG 接入

## 适合用 LangChain 的场景

- 快速验证一个 Agent 想法
- 系统复杂度还不高
- 希望先把产品能力跑通
- 暂时不需要自己下沉到状态机编排层
