## 怎么用这部分

- 阶段学习时，把它当“回查资料库”。
- 做项目时，把它当“专题知识支撑库”。
- 做课后项目时，如果你发现某个概念理解不稳，就回到对应主题目录补课。
- 如果你想知道某个知识点在官方 LangChain Docs 里对应哪一页，也从这里进入。
