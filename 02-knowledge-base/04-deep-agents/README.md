# Deep Agents

这一部分聚焦 Deep Agents 作为复杂任务型 Agent 的高层入口。

## 为什么要学 Deep Agents

- 它更接近很多人心目中的“真正执行型助理”
- 它让你更早面对 planning、subagents、context engineering 这些复杂问题
- 它适合复杂目标、长任务和多阶段任务执行

## 这里会覆盖什么

- Planning
- Context engineering
- Subagents
- Human approval
- 复杂任务治理
- 长任务执行

## 适合用 Deep Agents 的场景

- 深度研究
- 多阶段内容策划
- 复杂代码与文档工作
- 需要任务分解和子代理协作的场景
