# Deep Agents

## 官方定位

根据官方 overview，Deep Agents 是构建复杂 agent 的高层入口之一，重点在于规划、上下文工程、子代理协作、复杂任务执行和更强的任务完成能力。

参考:

- [Deep Agents overview](https://docs.langchain.com/oss/python/deepagents/overview)
- [Customization](https://docs.langchain.com/oss/python/deepagents/customization)
- [Context engineering](https://docs.langchain.com/oss/python/deepagents/context-engineering)

## 为什么要学 Deep Agents

- 它更接近很多人心目中的“真正智能执行助手”
- 它让你更早看到复杂任务 Agent 的组织方式
- 它让你开始认真面对规划、子代理和长期任务的问题

## 这部分的核心知识点

- 复杂任务如何被规划
- 什么是 context engineering
- subagents 什么时候是必要的
- 人类审批与复杂行为治理
- 如何避免复杂 Agent 失控

## 适合用 Deep Agents 的场景

- 深度研究
- 多阶段内容策划
- 复杂任务分解与执行
- 代码和文档型复杂知识工作

## 新手最值得注意的点

- Deep Agents 不是“更高级的自动提示词”
- 它真正难的是任务边界、上下文组织和行为治理
- 如果底层心智模型不稳，Deep Agents 很容易被误用成“复杂但不可控”
