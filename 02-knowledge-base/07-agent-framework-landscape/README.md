# Agent Framework Landscape

这套课程主线围绕 LangChain 生态，但不会把你困在单一框架里。

## 为什么要学框架地图

- 真正做项目时，你需要做框架选择
- 不同框架适合不同复杂度、团队背景和产品目标
- 学会比较，才能真正理解 LangChain 体系的价值与边界

## 课程会纳入比较的主流框架

- LangChain
- LangGraph
- Deep Agents
- OpenAI Agents SDK
- PydanticAI
- CrewAI
- AutoGen

## 比较维度

- 上手速度
- 抽象层级
- 状态与工作流能力
- 复杂任务能力
- 工具与协议接入
- 前后端整合难度
- 生产化与治理能力
