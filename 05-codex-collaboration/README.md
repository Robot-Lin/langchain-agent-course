# 与 Codex 的协作系统

这个目录定义课程中的学习协作模式。你在这套课程里，不只是“让 Codex 帮你写代码”，而是把 Codex 当成：

- 课程搭档
- 概念解释器
- 项目拆解器
- 原型搭建助手
- 评审与复盘伙伴

## 推荐阅读

- [session-patterns.md](X:/AI%20work/LangChain%20Tutorial/05-codex-collaboration/session-patterns.md)

## 这部分解决什么问题

- 如何向 Codex 提需求
- 如何让 Codex 先解释、再搭建、再评审
- 如何避免把学习过程变成“直接索要答案”
- 如何在 IDE 环境里做真正的共学与共创
