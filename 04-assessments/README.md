# 评估系统

这套课程不使用“只看你记住了多少名词”的评估方式，而是使用更接近真实项目能力的评价框架。

## 评估维度

- 概念理解
- 架构解释
- 项目拆解
- 工具与知识接入设计
- 状态与控制流设计
- 体验设计
- 风险识别
- 复盘能力

## 评估形式

- 概念口述
- 架构图复述
- 项目方案评审
- 原型演示
- 全栈联调演示
- 失败案例复盘
- 框架选型说明

## 评分原则

- 能不能说清楚
- 能不能做取舍
- 能不能把复杂系统讲成别人听得懂的东西
- 能不能识别风险和边界
- 能不能把思路迁移到新场景

## Codex 在评估里的角色

- 让 Codex 充当导师
- 让 Codex 充当评审
- 让 Codex 充当反方，专门指出方案漏洞
- 让 Codex 帮你生成下一轮训练任务
