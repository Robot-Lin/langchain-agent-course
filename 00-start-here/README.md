# Start Here

这是课程的第一入口。你不需要先掌握全部知识，也不需要一开始就写代码。你需要做的是先建立一个正确的学习姿势。

## 这套课程怎么学

- 在 IDE 中与 Codex 配合完成学习
- 以问答、拆题、设计、复盘、改进为主
- 通过 Vibe Coding 逐步把想法变成原型
- 每一阶段都同时学习概念、结构、边界和实战
- 每一阶段都有项目和考核，不是只看概念

## 第一周最重要的三件事

- 理解 Agent 不是“一个提示词”，而是一个包含模型、上下文、工具、状态、记忆、决策和交互的系统
- 理解 LangChain、LangGraph、Deep Agents 三者的层次关系
- 建立与 Codex 的标准协作流程，避免陷入“只会让 AI 生成代码”的假熟练

## 你将怎么和 Codex 配合

- 你负责描述目标、偏好、场景、限制
- Codex 负责梳理结构、落地资料、生成框架、推进项目
- 你通过提问、复述、选择方案、要求解释来巩固理解
- 所有阶段都要求你能说清楚“为什么这样设计”

## 建议的学习节奏

- Session 1: 看路线图，确定你的目标场景
- Session 2: 建立 Agent 基础心智模型
- Session 3: 先做一个最小可用 Agent 教学项目
- Session 4 以后: 进入 LangChain、LangGraph、Deep Agents 分阶段深入

## 对自己的最低要求

- 每学完一个概念，都能用自然语言复述
- 每做完一个项目，都能说出它的输入、输出、状态、工具、风险和体验设计
- 每完成一个阶段，都能回答“我为什么在这里用这个框架，而不是另一个”
