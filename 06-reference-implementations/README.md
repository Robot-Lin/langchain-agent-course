  负责训练项目与项目协作路线
- `06-reference-implementations/`
  负责提供可以直接读代码、跑起来、对照主路径理解的参考实现

## 建议怎么用

如果你现在的目标是：

- 想先看“一个最小可运行项目到底怎么写”：从这里开始
- 想对照课程读代码：把这里和对应阶段讲义一起看
- 想学文件职责、主路径、调用链：优先读每个参考实现自带的 README 和读码指南

## 推荐阅读顺序

1. 先看项目内的 `README.md`
2. 再看 `CODE_READING_GUIDE.md`（如果有）
3. 再顺着 `schema -> tool -> agent -> workflow -> main` 去读
