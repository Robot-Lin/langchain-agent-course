# 课程入口

这个目录承载整套教学系统的课程主框架。你可以把它理解为课程地图、阶段路线图和阶段放行标准的集中入口。

## 先看什么

建议按下面顺序阅读：

1. [course-roadmap.md](X:/AI%20work/LangChain%20Tutorial/01-curriculum/course-roadmap.md)
2. [learning-loop.md](X:/AI%20work/LangChain%20Tutorial/01-curriculum/learning-loop.md)
3. [stages/README.md](X:/AI%20work/LangChain%20Tutorial/01-curriculum/stages/README.md)
4. [competency-matrix.md](X:/AI%20work/LangChain%20Tutorial/01-curriculum/competency-matrix.md)
5. [stage-gates.md](X:/AI%20work/LangChain%20Tutorial/01-curriculum/stage-gates.md)

## 这里包含什么

- `course-roadmap.md`
  课程整体路线图与 8 个阶段的目标
- `learning-loop.md`
  每个阶段统一的学习闭环
- `fullstack-agent-path.md`
  课程如何通向全栈 Agent 项目
- `stages/`
  `Stage 1-8` 的完整教学包入口
- `competency-matrix.md`
  学习者能力矩阵
- `stage-gates.md`
  阶段放行与毕业标准

## 推荐使用方式

- 如果你是第一次进入课程，从路线图开始。
- 如果你正在按阶段学习，直接进入 `stages/`。
- 如果你想判断自己是否可以进入下一阶段，优先看 `stage-gates.md`。
