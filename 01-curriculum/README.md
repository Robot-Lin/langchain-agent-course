# 课程入口

这个目录承载整套教学系统的课程主框架。你可以把它理解为课程地图、阶段路线图、学习闭环和阶段放行标准的集中入口。

## 先看什么

建议按下面顺序阅读：

1. [course-roadmap.md](./course-roadmap.md)
2. [learning-loop.md](./learning-loop.md)
3. [stages/README.md](./stages/README.md)
4. [competency-matrix.md](./competency-matrix.md)
5. [stage-gates.md](./stage-gates.md)

## 这里包含什么

- `course-roadmap.md`
  课程整体路线图与 8 个阶段的目标
- `learning-loop.md`
  每个阶段统一的学习闭环
- `fullstack-agent-path.md`
  课程如何通向全栈 Agent 项目
- `stages/`
  `Stage 1-8` 的完整教学包入口
- `competency-matrix.md`
  学习者能力矩阵
- `stage-gates.md`
  阶段放行与毕业标准

## 当前课程总入口已经对齐到什么状态

- 课程顺序已经按新版官方架构统一：
  先 `LangChain`，再 `LangGraph`，再 `Deep Agents`，并在后段补齐 `LangSmith`
- 从 `Stage 2` 开始，课程不再只讲概念，而是逐步加入代码示例、API 对照和全项目阅读
- 学习者的目标不只是把项目做出来，还要能讲清整个项目的目录、文件、主路径、关键逻辑和框架映射
- 项目教学、阶段放行和 Codex 对练都已经围绕“全项目理解”做了校准

## 这个目录现在额外强调什么

这套课程现在已经明确加入一条主线：

- 学习者不只要会描述需求
- 不只要会和 AI 协作做出 Agent
- 还要能看懂关键代码、解释关键逻辑、做基本审核判断

所以这里的路线图、学习闭环、能力矩阵和阶段放行标准，已经都围绕这条要求做了统一。

## 推荐使用方式

- 如果你是第一次进入课程，从路线图开始。
- 如果你正在按阶段学习，直接进入 `stages/`。
- 如果你想判断自己是否可以进入下一阶段，优先看 `stage-gates.md`。
- 如果你想确认这套课程要求你读懂哪些代码，优先看 `learning-loop.md` 和 `competency-matrix.md`。
- 如果你想确认代码教材已经推进到哪一层，优先进入 `Stage 2` 之后的各阶段 `README.md`、`02-concepts.md`、`03-methods.md` 和 `10-full-project-reading.md`。
