# Stage 4 教学包

这是课程中从“可控工作流”进入“复杂任务型 Agent”的关键单元。它承接 Stage 3，把学习者带入 Deep Agents 的复杂任务组织方式。

## 这一阶段学完之后应该做到什么

- 能解释 Deep Agents 适合什么问题，不适合什么问题
- 能理解规划、context engineering、subagents 和复杂任务治理的关系
- 能设计一个复杂任务型 Agent 的角色分工与上下文层次
- 能说明为什么这个任务更适合 Deep Agents，而不是普通 LangGraph 工作流

## 本阶段的核心问题

- 什么任务已经超出普通工作流的舒适区
- 规划为什么不是“多想几步”这么简单
- context engineering 为什么不是“塞更多上下文”
- subagents 什么时候必要，什么时候只是制造复杂度
- 复杂任务 Agent 如何避免失控

## 对整个课程的作用

Stage 4 训练的是:

- 复杂任务拆解能力
- 上下文组织能力
- 角色分工能力
- 深层任务治理意识
- 面向复杂目标的 Agent 设计能力
