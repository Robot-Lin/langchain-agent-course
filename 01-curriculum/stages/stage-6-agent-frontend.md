# Stage 6: Agent Frontend

## 阶段目标

- 把 Agent 从“有后端能力”推进到“有真实产品体验”。
- 学会设计用户可理解、可控制、可审批的 Agent 前端。
- 建立前端与 Agent runtime 的协同意识。

## 学习重点

- Chat UX 与 Agent UX
- 流式输出
- 任务计划可视化
- 工具执行状态展示
- 审批界面
- 历史记录与来源可见性
- 失败反馈与解释感

## 这章真正要学会什么

不是“会做一个聊天界面”，而是能回答：

- 用户如何知道 Agent 正在做什么。
- 用户如何知道什么时候该审批或接管。
- 用户如何稳定地阅读结果而不是消化一整段文本。
- 复杂后端能力如何被前端翻译成可信、可控、可理解的产品体验。
