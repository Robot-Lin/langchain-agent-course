# Stage 2 教学包

这是课程中第一套真正进入框架实战的教学单元。它承接 Stage 1 的系统判断，把学习者带入 LangChain 的高层 Agent 构建。

## 这一阶段学完之后应该做到什么

- 能用 LangChain 搭出一个高层 Agent 原型
- 能理解 tools、structured output、middleware、HITL 在高层 Agent 中的作用
- 能判断哪些需求还适合停留在 LangChain 高层抽象
- 能把一个 Stage 1 的自然语言设计推进成可工作的原型方案

## 本阶段的核心问题

- LangChain 到底帮我们抽象了什么
- 什么是“高层 Agent 原型”
- 工具、结构化输出、middleware 为什么是高层开发的关键
- 人工审核为什么可以在高层就先接进去
- 什么情况应该继续用 LangChain，什么情况应该转向 LangGraph

## 对整个课程的作用

Stage 2 是用户第一次真正把 Agent 设计推向“可搭建、可演示、可验证”的关键阶段。

它训练的是:

- 组件化思维
- 高层原型搭建能力
- 输出和工具边界设计
- 从概念到原型的推进能力
