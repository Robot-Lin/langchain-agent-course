# 07 Codex Workshop

这一部分是给学习者在 Stage 2 直接拿来用的 Codex 对练脚本。

## 模式一: 高层原型讲解

可以对 Codex 说:

- 用课程学习者能听懂的方式解释 LangChain 为什么适合高层 Agent 原型
- 结合一个真实项目解释 structured output 的价值
- 比较 prompt 控制和 middleware 控制的差别

## 模式二: 原型搭建陪练

可以对 Codex 说:

- 我想把这个需求做成 LangChain 原型，先别写完整代码，先帮我定主任务、工具和输出结构
- 先只保留一个工具，不要过度设计
- 每一步都解释这一步对应哪一个 Stage 2 知识点

## 模式三: 结构化输出设计

可以对 Codex 说:

- 帮我为这个 Agent 设计 structured output
- 每个字段都告诉我它是给人看的，还是给系统用的

## 模式四: 高层边界评审

可以对 Codex 说:

- 站在架构评审角度，判断这个项目还适不适合继续留在 LangChain 高层
- 哪些信号说明我应该开始用 LangGraph

## 模式五: 课后项目导师模式

可以对 Codex 说:

- 我现在做 Stage 2 课后项目，请不要直接给完整答案
- 先问我问题，检查我的主任务、工具、结构化输出和控制逻辑
- 最后按验收标准帮我评分

## 推荐学习动作

1. 先让 Codex 帮你压缩成高层原型
2. 再自己定义工具和输出
3. 再让 Codex 查缺补漏
4. 最后让 Codex 判断是否已经要进入 LangGraph
