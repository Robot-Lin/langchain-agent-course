# Stage 7: Full-stack Delivery

## 阶段目标

- 把前端、后端、Agent runtime、数据层和知识层组装成完整产品。
- 学会部署、观测、评测和版本迭代。
- 具备做全栈 Agent 项目交付的能力。

## 关联官方主线

- [LangSmith Deployment](https://docs.langchain.com/langsmith/deployment)
- [Agent Server](https://docs.langchain.com/langsmith/agent-server)
- [Agent Server API reference](https://docs.langchain.com/langsmith/server-api-ref)
- [Streaming API](https://docs.langchain.com/langsmith/streaming)
- [LangSmith Observability](https://docs.langchain.com/langsmith/observability)
- [LangSmith Evaluation](https://docs.langchain.com/langsmith/evaluation)

## 学习重点

- Full-stack architecture
- 前后端通信
- Agent Server
- Streaming integration
- Deployment
- Observability
- Evaluation
- 审计、权限和产品边界
- 框架选型与迁移

## 这章真正要学会什么

不是“会把组件接起来”，而是能回答：

- 一个 Agent 产品从用户请求到系统结果的完整路径是什么。
- 这个系统如何部署、运行和追踪。
- 这个系统如何被评估并决定是否可交付。
- 为什么当前架构适合当前产品阶段。
