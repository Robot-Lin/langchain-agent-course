# 06 Review Checklist

## 自测清单

如果下面任意 3 项做不到，建议不要急着进入 Stage 7。

- 我能解释 Agent UX 和聊天 UX 的差别
- 我能设计一个状态化界面分区
- 我能说明 streaming 在产品中的价值
- 我能设计结构化结果展示
- 我能设计审批界面
- 我能说明历史与来源感为什么重要
- 我能把复杂后端能力翻译成前端体验
- 我能说明为什么前端会影响系统可信度

## 阶段答辩题

### 必答题

1. 为什么 Agent 产品不能只有聊天框
2. 为什么 streaming 在 Agent 系统中不仅是视觉效果
3. 为什么审批交互必须被前端明确表达
4. 为什么结构化输出对前端特别重要

### 进阶题

1. 如果用户看不见当前状态，会出现什么问题
2. 为什么历史、来源和结果分区会影响信任感
3. 你会如何把 Stage 5 的工程化 Agent 做成一个可用产品界面

## 放行标准

学习者至少要做到:

- 完成 1 个 Agent 前端结构设计
- 完成 1 个跟学项目
- 完成 1 个课后独立项目
- 完成 1 次围绕状态、审批和展示结构的答辩或评审

## 进入 Stage 7 前的最后确认

只有当学习者已经能把系统能力翻译成产品体验，进入全栈交付层才会真正有意义。
