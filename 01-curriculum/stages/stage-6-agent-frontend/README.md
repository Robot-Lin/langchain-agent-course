# Stage 6 教学包

这是课程中把 Agent 从“后台系统”推进到“真正可用产品体验”的关键单元。它承接 Stage 5，把学习者带入 Agent UX、状态可视化、流式交互和审批界面设计。

## 这一阶段学完之后应该做到什么

- 能解释 Agent UX 和普通聊天界面的差别
- 能设计适合 Agent 的流式、状态化和可审批交互
- 能让用户看见 Agent 正在做什么、将要做什么和哪里需要接管
- 能设计一个支持复杂任务的前端交互框架

## 本阶段的核心问题

- 为什么 Agent 产品不能只有聊天框
- 为什么 streaming、structured output 和审批界面会直接影响可用性
- 为什么复杂任务必须有状态可视化和历史可见性
- 为什么前端是可信度、控制感和体验质量的核心部分

## 对整个课程的作用

Stage 6 训练的是:

- Agent UX 思维
- 状态可视化能力
- 审批交互设计能力
- 前后端体验衔接能力
- 产品可控性设计能力
