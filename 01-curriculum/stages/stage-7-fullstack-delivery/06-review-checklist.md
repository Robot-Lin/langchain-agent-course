# 06 Review Checklist

## 自测清单

如果下面任意 3 项做不到，建议不要急着进入 Stage 8。

- 我能画出全栈 Agent 的完整链路
- 我能解释 Agent Server 的交付角色
- 我能说明流式更新如何贯穿前后端
- 我能写出部署与运行思路
- 我能写出交付验收指标
- 我能说明 observability 和 evaluation 如何进入产品闭环
- 我能做一次框架选型说明
- 我能把 Agent 从工程方案推进到产品交付

## 阶段答辩题

### 必答题

1. 为什么全栈 Agent 交付不只是“前端加后端”
2. Agent Server 为什么对交付层重要
3. 为什么 deployment、observability、evaluation 属于产品交付的一部分
4. 为什么一个系统能演示不等于它已经可以交付

### 进阶题

1. 如果一个产品链路里没有流式反馈，会损失什么
2. 为什么选型说明是交付的一部分
3. 你会如何把 Stage 6 的前端和 Stage 5 的工程系统真正打通

## 放行标准

学习者至少要做到:

- 完成 1 个全栈 Agent 产品交付设计
- 完成 1 个跟学项目
- 完成 1 个课后独立项目
- 完成 1 次围绕交付链路、部署和评测的答辩或评审

## 进入 Stage 8 前的最后确认

只有当学习者已经能把系统完整交付，毕业项目阶段才不会只是“再做一个更大的 Demo”。
