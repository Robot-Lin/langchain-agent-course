# 07 Codex Workshop

这一部分是给学习者在 Stage 7 直接拿来用的 Codex 对练脚本。

## 模式一: 全链路讲解

可以对 Codex 说:

- 用一个真实产品解释全栈 Agent 从前端到运行时的完整链路
- 讲讲 Agent Server、deployment、streaming、evaluation 在里面各自负责什么
- 帮我比较“能演示”和“可交付”的差别

## 模式二: 系统组装陪练

可以对 Codex 说:

- 帮我把这个 Agent 产品组装成完整全栈链路
- 每一步都说明对应的是哪个 Stage 7 知识点
- 优先指出我缺的是哪一层

## 模式三: 交付评审

可以对 Codex 说:

- 站在产品交付评审角度，指出这个系统还缺什么才能上线或演示
- 帮我补部署、追踪和评测部分

## 模式四: 选型评审

可以对 Codex 说:

- 帮我写一段框架和交付方式的选型说明
- 指出如果团队更小或用户量更大时，架构会怎么调整

## 模式五: 课后项目导师模式

可以对 Codex 说:

- 我现在做 Stage 7 课后项目，请不要直接给我完整答案
- 你只负责检查我的全链路设计、部署、评测和交付说明
- 最后按验收标准帮我评分

## 推荐学习动作

1. 先画完整链路
2. 再补部署与追踪
3. 再补验收与评测
4. 最后让 Codex 做交付评审
