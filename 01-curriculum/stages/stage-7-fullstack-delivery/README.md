# Stage 7 教学包

这是课程里把“前端、后端、Agent、数据、观测”真正组装成可交付产品的关键单元。它承接 Stage 6，把学习者带入全栈 Agent 交付、部署、运行、评测、产品化和迭代。

## 学完这一章之后应该做到什么

- 能把前端、后端、Agent runtime、数据层、知识层和评测层组装成完整系统。
- 能解释 Agent Server、deployment、streaming integration 在交付中的角色。
- 能说明全栈交付不只是技术拼装，而是产品链路设计。
- 能做出一个可演示、可追踪、可验证、可迭代的全栈 Agent 原型。

## 本章的核心问题

- 一个 Agent 产品从用户点击到系统返回结果，完整链路是什么。
- Agent Server 和普通 API 层的关系是什么。
- 为什么 deployment、observability 和 evaluation 是交付问题的一部分。
- 为什么真正的全栈 Agent 必须可演示、可追踪、可验证、可迭代。
- 如何做框架选型，而不是只沿用惯性。

## 本章在整套课程里的作用

Stage 7 训练的是交付视角：

- 你开始把系统看成完整产品链路，而不只是前后端若干模块。
- 你开始关心运行、演示、排障、评测和版本迭代。
- 你开始具备把前面 1 到 6 章真正组装成产品的能力。
