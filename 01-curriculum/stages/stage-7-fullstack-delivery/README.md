# Stage 7 教学包

这是课程中把“前后端、Agent、数据、观测”真正组装成可交付产品的关键单元。它承接 Stage 6，把学习者带入全栈 Agent 交付、部署、评测、产品化和迭代。

## 这一阶段学完之后应该做到什么

- 能把前端、后端、Agent runtime、数据层、知识层和评测层组装成完整系统
- 能解释 Agent Server、deployment、streaming integration 的角色
- 能说明全栈交付不只是技术拼装，而是产品链路设计
- 能做一个可演示、可追踪、可迭代的全栈 Agent 原型

## 本阶段的核心问题

- 一个 Agent 产品从用户点击到系统返回结果，完整链路是什么
- Agent Server 和普通 API 层的关系是什么
- 为什么 deployment、observability 和 evaluation 是交付问题的一部分
- 为什么真正的全栈 Agent 需要“可演示、可追踪、可重放、可迭代”
- 如何做框架选型，而不是只沿用惯性

## 对整个课程的作用

Stage 7 训练的是:

- 全栈组装能力
- 交付链路思维
- 部署与运行思维
- 产品化与迭代思维
- 框架选型与工程取舍能力
