# 05 Challenges

## Challenge 1: 选题练习

请提出 3 个你可能做的毕业项目方向，并分别说明:

- 用户是谁
- 为什么真的需要 Agent
- 它将覆盖课程哪些能力层

## Challenge 2: 覆盖检查练习

对你选中的题目，逐项检查它是否覆盖:

- 系统判断
- Agent 主线
- 状态或工作流
- 工程化层
- 前端体验层
- 交付与评测层

## Challenge 3: 演示路径练习

请为你的毕业项目写出:

- 一个 3 分钟演示版本
- 一个 10 分钟完整讲解版本

## Challenge 4: 风险答辩练习

请提前回答:

- 这个项目哪里最容易失控
- 哪个设计最可能被评审质疑
- 哪个部分最需要下一版迭代

## Challenge 5: 方法迁移练习

请再选一个完全不同的场景，用 10 句话说明如果你来做，会怎么复用这套方法。
