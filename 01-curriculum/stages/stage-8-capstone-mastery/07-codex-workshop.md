# 07 Codex Workshop

这一章里，Codex 最适合扮演的不是“帮你把毕业项目做完”，而是选题陪练、答辩评审员和复盘教练。

## Codex 在 Stage 8 里最适合的 5 个角色

### 1. 选题比较器

适合帮你比较多个毕业方向：

- 哪个更符合课程目标
- 哪个覆盖度更高
- 哪个风险更可控

推荐提问：

```text
请你帮我比较这 3 个毕业项目方向，重点从课程能力覆盖、风险、主路径清晰度和可答辩性来评估。不要直接替我决定，只帮我做对比。
```

### 2. 综合架构评审员

适合在你已经有初版方案后，让 Codex 检查它是否真的覆盖前 7 章能力。

推荐提问：

```text
请你检查这个毕业项目是否真的覆盖了前 7 个阶段的核心能力。优先指出哪些地方还只是单点能力，哪些地方已经形成闭环。
```

### 3. 答辩模拟器

适合让 Codex 扮演评审，追问你而不是直接给答案。

推荐提问：

```text
你来扮演毕业评审，按 Stage 8 的答辩题逐条追问我。不要直接给答案，先让我回答，再指出我哪里解释得不够清楚。
```

### 4. 复盘教练

适合让 Codex 帮你把复盘写得更有判断力。

推荐提问：

```text
请你帮我整理这次毕业项目复盘，重点指出哪些设计有效、哪些设计过度、哪些问题还没解决，以及下一版应该怎么排优先级。
```

### 5. 方法迁移陪练

适合让 Codex 给你一个新场景，看你能否把方法迁移过去。

推荐提问：

```text
请给我一个全新的 Agent 场景，让我用这套课程方法快速设计。你负责做评审，指出我哪里迁移得还不够好。
```

## 推荐学习顺序

1. 先自己定题。
2. 再让 Codex 做覆盖和风险比较。
3. 先自己画综合架构。
4. 再让 Codex 做闭环评审。
5. 先自己准备答辩。
6. 最后让 Codex 做模拟评审和复盘反馈。

## 常见错误用法

### 错误一

让 Codex 直接帮你决定毕业题目。

问题：

- 你会跳过最重要的独立取舍训练。

### 错误二

让 Codex 直接写完整答辩答案。

问题：

- 你会失去最关键的表达和应对质疑训练。

### 错误三

把复盘也交给 Codex 代写。

问题：

- 你会看起来有总结，但并没有形成真正的方法内化。

## Stage 8 推荐协作闭环

- 你先选题。
- 让 Codex 做方向比较。
- 你再画综合架构。
- 让 Codex 做覆盖与闭环评审。
- 你再准备答辩与复盘。
- 让 Codex 做模拟评审和迁移训练。

## 一次完整对练模板

```text
我正在学习 Stage 8: Capstone Mastery。
这是我的毕业项目方向和当前综合方案。
请你不要直接替我做完整答案。
先按以下顺序帮助我：
1. 检查这个题目是否真的值得做，以及是否覆盖课程核心能力。
2. 检查我的综合架构哪里还没有形成闭环。
3. 站在毕业评审角度，指出最容易被追问的 5 个问题。
4. 帮我检查复盘和下一版规划是否有判断力。
5. 最后只告诉我下一轮该优先补什么。
```
