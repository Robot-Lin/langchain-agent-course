# 07 Codex Workshop

这一部分是给学习者在 Stage 8 直接拿来用的 Codex 对练脚本。

## 模式一: 选题讨论

可以对 Codex 说:

- 帮我比较这 3 个毕业项目方向，哪个最适合当前课程目标
- 不要直接替我决定，先帮我做覆盖分析和风险分析

## 模式二: 综合架构评审

可以对 Codex 说:

- 帮我检查这个毕业项目是否真的覆盖了前 7 个阶段的核心能力
- 指出哪里还只是单点能力，哪里已经形成闭环

## 模式三: 答辩陪练

可以对 Codex 说:

- 你来当评审，按毕业答辩题逐条追问我
- 不要直接给答案，先让我回答，再指出漏洞

## 模式四: 复盘陪练

可以对 Codex 说:

- 帮我把这次毕业项目复盘写清楚
- 重点指出哪些设计有效，哪些设计过度，下一版应该怎么排优先级

## 模式五: 方法迁移训练

可以对 Codex 说:

- 给我一个新场景，让我用这套方法快速设计
- 你负责扮演评审，指出我哪里迁移得不够好

## 推荐学习动作

1. 先定题
2. 再画综合架构
3. 再准备答辩
4. 最后做复盘和迁移训练
