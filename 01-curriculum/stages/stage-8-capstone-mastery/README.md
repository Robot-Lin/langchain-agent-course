# Stage 8 教学包

这是整套课程的毕业项目与掌握闭环阶段。它承接 Stage 1 到 Stage 7，把学习者带入“独立完成一个从后端到前端的 Agent 产品，并能清楚解释、评审、复盘和迭代”的完整过程。

## 学完这一章之后应该做到什么

- 能独立选择一个值得做的 Agent 产品方向。
- 能把前 7 个阶段的能力整合进一个完整产品。
- 能完成架构说明、交付说明、演示、答辩和复盘。
- 能把这套方法迁移到新的场景，而不是只会做一个项目。

## 本章的核心问题

- 这个毕业项目为什么值得做。
- 它是否真的覆盖了课程核心能力层。
- 它是否已经从 Demo 走向可解释、可评审、可迭代的产品。
- 学习者是否真正掌握了方法，而不只是完成了一个特定项目。

## 本章在整套课程里的作用

Stage 8 训练的是最终掌握：

- 独立选题能力
- 综合架构能力
- 产品表达与答辩能力
- 评审与复盘能力
- 方法迁移能力
