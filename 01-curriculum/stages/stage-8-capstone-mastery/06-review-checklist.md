# 06 Review Checklist

## 自测清单

如果下面任意 3 项做不到，说明还不算真正毕业。

- 我能清楚说明我的选题价值
- 我能画出综合系统图
- 我能讲清用户旅程和主路径
- 我能解释为什么选这条 Agent 主线
- 我能说明前端、后端和 runtime 如何协同
- 我能说明评测、追踪和治理设计
- 我能做一场完整答辩
- 我能写出有价值的复盘和下一版规划

## 毕业答辩题

### 必答题

1. 这个项目为什么值得做
2. 为什么它需要 Agent，而不是普通应用
3. 为什么你选择当前架构
4. 这个系统如何从前端到后端形成完整闭环
5. 这个系统如何被评测、追踪和治理

### 进阶题

1. 哪个设计是你最犹豫的取舍
2. 如果用户规模扩大或团队缩小，架构会怎么改
3. 如果再给你两周，你最优先做什么
4. 如何把这套方法迁移到另一个行业

## 毕业验收标准

学习者至少要做到:

- 完成 1 个完整毕业项目方案
- 完成 1 份系统图和用户旅程说明
- 完成 1 次完整答辩
- 完成 1 份复盘报告
- 完成 1 份下一版规划

## 真正掌握的判断

如果学习者不仅能展示当前项目，还能把方法迁移到新场景，就说明这套课程真正完成了它的目标。
