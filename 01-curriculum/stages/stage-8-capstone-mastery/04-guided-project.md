# 04 Guided Project

本阶段引导项目不是固定题目，而是 `毕业项目框架引导`。学习者要从已有项目方向中选择一个，升级为完整毕业项目。

## 项目目标

选择一个自己最想做、也最能覆盖课程能力的 Agent 产品方向，完成从题目定义到综合设计、演示、答辩和复盘的完整过程。

## 为什么这一阶段不再给单一固定题目

因为毕业阶段要验证的是:

- 选题能力
- 取舍能力
- 综合能力

如果题目完全固定，反而很难验证这些能力。

## Step 1: 选择毕业方向

可选方向例如:

- 深度研究助手
- 学习教练 Agent
- 企业流程审批助手
- 内容工作台
- 个人操作系统型 Agent

## Step 2: 画综合架构

至少包括:

- 前端
- Agent runtime
- 知识或工具层
- 数据层
- 评测与追踪层
- 审批与治理层

## Step 3: 定义主路径与演示路径

明确:

- 用户如何进入
- 系统如何完成核心任务
- 演示时看什么

## Step 4: 准备答辩材料

至少准备:

- 选题说明
- 系统图
- 用户旅程
- 风险与边界
- 下一版规划

## Step 5: 做毕业复盘

项目做完后，必须回答:

- 哪些能力是真正掌握的
- 哪些能力仍需要补强
