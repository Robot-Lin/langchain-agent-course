# 05 Challenges

## Challenge 1: 边界判断

请分别判断下面需求更偏向:

- memory
- RAG
- 数据库
- MCP / 工具接入

### 题目 A

记住用户偏好的回答风格。

### 题目 B

检索公司制度文档后回答问题。

### 题目 C

记录审批状态和执行日志。

### 题目 D

调用外部系统获取实时工单数据。

## Challenge 2: 分层架构练习

请为一个 Agent 系统画出分层架构，并至少包括:

- Agent 层
- 工具或 MCP 层
- 知识层
- 数据层
- API 层
- 观测层
- 治理层

## Challenge 3: 指标设计练习

请为一个知识问答 Agent 写出:

- 3 个运行指标
- 3 个质量指标
- 3 个失败信号

## Challenge 4: 治理练习

假设一个 Agent 可以:

- 读知识库
- 查客户数据
- 修改工单
- 对外发送消息

请为它写出最小权限治理清单。

## Challenge 5: 工程化复盘练习

拿一个你前面阶段做过的 Agent，回答:

- 它还缺哪些工程化层
- 如果现在要接真实系统，最大风险在哪里
