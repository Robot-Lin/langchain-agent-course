# Stage 8: Capstone Mastery

## 阶段目标

- 完成一个从后端到前端的毕业级 Agent 项目。
- 形成可迁移的系统设计与产品迭代能力。
- 建立独立构建、答辩、复盘和迁移的方法。

## 毕业项目要求

- 必须包含 Agent runtime
- 必须包含工具或 MCP 接入
- 必须包含知识系统或数据层
- 必须包含前端交互
- 必须包含观测和评测
- 必须包含一次复盘与下一步规划

## 推荐毕业方向

- 深度研究助手
- 学习教练 Agent
- 企业流程审批助手
- 创作者工作台
- 个人操作系统型 Agent

## 完成路径

### 1. 选题

- 用户是谁
- 高频任务是什么
- 为什么需要 Agent

### 2. 架构

- 选择 LangChain、LangGraph 或 Deep Agents 主线
- 定义工具、知识、状态、记忆和审批机制

### 3. 原型

- 先做最小工作版本
- 先打通主路径

### 4. 产品化

- 补上前端体验
- 补上观测与评测
- 补上部署与风险边界

### 5. 复盘

- 什么工作得好
- 什么失控了
- 下一步如何迭代

## 毕业答辩要求

- 口头解释完整架构
- 演示主路径
- 展示失败路径或审批路径
- 展示评测思路
- 说明为什么这个设计值得保留

## 掌握标准

- 不是只会做一个项目
- 而是能把同样方法迁移到新场景
