# Stage 1 教学包

这是课程中第一套真正完整的教学单元。它不是让学习者先学 API，而是先建立判断力。

## 这一阶段学完之后应该做到什么

- 能说清 Agent 到底是什么
- 能判断一个需求是否真的需要 Agent
- 能把一个模糊需求拆成系统结构
- 能识别工具、状态、记忆、审批、反馈在系统中的位置
- 能在不写代码的前提下设计一个最小可行 Agent

## 本阶段的核心问题

- Agent 和普通聊天机器人到底差在哪
- 什么情况需要工具，什么情况不需要
- 什么情况需要状态和记忆
- 什么情况必须设置人工确认
- 为什么很多“看起来很聪明”的系统其实不适合上线

## 对整个课程的作用

如果这一阶段没有打牢，后面学 LangChain、LangGraph 和 Deep Agents 很容易变成只会堆功能，不会做取舍。

这一步真正训练的是:

- 系统视角
- 边界意识
- 需求拆解能力
- 最小可行设计能力
