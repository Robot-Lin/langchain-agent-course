## Code Reading Track

从现在起，这里的每个阶段都不只训练“做出来”，还训练：

- 把知识点对应到功能
- 把功能对应到文件和代码
- 解释一条主路径如何从输入走到输出
- 对 AI 写出的代码做基本审核判断
